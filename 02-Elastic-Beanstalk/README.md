# 🌱 Task 02 – Deploying a Scalable Web Application Using AWS Elastic Beanstalk

## Overview

Amazon Elastic Beanstalk is an easy-to-use service for deploying and scaling web applications. It supports Java, .NET, PHP, Node.js, Python, Ruby, Go, and Docker — and handles the underlying infrastructure for you automatically.

---

## What I Learned

- How Elastic Beanstalk simplifies app deployment
- The difference between Web Server and Worker environments
- How to configure and launch a Beanstalk environment

---

## Key Concepts

| Concept | Description |
|---------|-------------|
| **Web Server Environment** | Runs a website or web API that serves HTTP requests |
| **Worker Environment** | Processes background tasks or scheduled jobs |
| **Platform** | The runtime your app uses (e.g., Node.js, Python) |
| **Environment** | An isolated deployment of your app with its own URL and resources |
