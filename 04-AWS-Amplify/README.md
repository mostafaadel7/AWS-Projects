# ⚡ Task 04 – Full Stack App Hosting with AWS Amplify

## Overview

AWS Amplify is a platform that lets you build, deploy, and host full-stack web and mobile applications on AWS. It provides frontend libraries, UI components, backend services, and CI/CD hosting — all in one place.

---

## What I Learned

- What AWS Amplify is and when to use it
- How Amplify Hosting works with Git repositories
- How CI/CD pipelines are automated through Amplify
- How Amplify connects to backend services (auth, database, storage)

---

## Key Concepts

| Concept | Description |
|---------|-------------|
| **Amplify Hosting** | Git-based hosting with automatic build & deploy on every push |
| **CI/CD Pipeline** | Every code push triggers automatic build and deployment |
| **CDN** | App is served globally through Amazon's content delivery network |
| **Backend** | Auth, database, and storage can be added with minimal config |
